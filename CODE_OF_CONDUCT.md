# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and abusive behavior are not
tolerated, in any project space (issues, PRs, discussions, chat).

## Reporting

Report unacceptable behavior to **conduct@uldren.com**. Reports are handled confidentially. Project
maintainers will review and respond, and may take any action they deem appropriate, up to and
including a ban from project spaces.

The full text of the Contributor Covenant 2.1 governs this project; see
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/> for the complete document,
including enforcement guidelines.
