Uldren Loom - multi-tab SQL over OPFS

Same-origin demo: this page, the loader, the engine, and the coordinator are all served from this one origin. (For the cross-origin "vendor CDN X + integrator Y" model, see integrator/.) Multiple tabs read+write one .loom: one tab is the host (runs the OPFS engine in a dedicated worker), a SharedWorker relays the others' ops to it.

 Same source in two tabs = shared; different sources = independent looms.
